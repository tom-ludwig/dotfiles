## Requirements

- Git
- GNU stow

## Installation

Checkout out the repositiory in your `$HOME` directory.

```
$ git clone https://github.com/activcoding/dotfiles.git
$ cd dotfiles
```

Then use GNU stow to create symlinks.

```
$ stow .
```
